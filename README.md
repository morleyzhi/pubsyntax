##Installation

Clone this git repo in Sublime Text's `Packages` directory.

### Mac OS X

```
~/Library/Application\ Support/Sublime\ Text\ 2/Packages/
```

### Windows

```
%APPDATA%/Sublime Text 2/Packages/
```
